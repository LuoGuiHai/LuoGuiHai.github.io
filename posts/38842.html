XPOSED 介绍和使用

前言

Xposed 框架(Xposed Framework)是一套开源的、在 Android 高权限模式下运行的框架服务，可以在不修改 APK 文件的情况下影响程序运行(修改系统)的框架服务，基于它可以制作出许多功能强大的模块，且在功能不冲突的情况下同时运作。

  • 中文名 Xposed 框架

  • 外文名 Xposed framework

  • 开发者 Rovo89

  • 网站 repo.xposed.info

  • 编程语言 C++、Java

  • 开发状态 已停更

应用介绍

优点

  1. 无需反编译 apk 文件达到修改 apk 的目的。

  2. 能在 odexed 或 deodexed 的 Rom 上运行。

应用介绍

当前，Per APP Setting(为每个应用设置单独的 dpi 或修改权限)、XPrivacy(防止隐私泄露)、对原生 Launcher 替换图标等应用或功能均基于此框架。

Xposed 并不支持 Android 9 以上版本，但可经由 EdXposed 代替

目前的已知问题：在一些手机的官方 Rom 中因为 oat 文件格式不同会导致卡在开机部分。Android 5 以上需要手动刷入框架。

那么接下来开始教程。

确认手机是否 root

原版 xposed 需要 root 权限，但也出现了不需要 root 就可以使用 xposed 模块的框架，比如太极、应用转生等。太极有模块版本限制，应用转生无模块限制，他们能使用大部分常用模块，但因为无 root 权限，所以都无法使用系统相关的模块。并且都需要卸载原应用程序重新安装，非常繁琐，升级应用也比较困难。 ROOT 教程 >>

无 root 太极举例

无 root 权限就使用太极【举例】，访问太极 官网 ，下载安装太极 app 并打开，无 root 权限将会是太极·阴。 7.0 蓝奏云链接备份

【以 QQ 举例】

  1. 点击右下角按钮，选择创建应用，等待手机已安装应用检测，选择 QQ 即可。太极·阴会重新签名 QQ，直接安装将会失败，需要卸载安装，请注意备份 QQ 数据。

这是无 root 使用 xposed 的通病，都需要卸载旧版安装重新签名的版本，数据无价！

  1. 一切完成后下载模块，进入模块管理界面，勾选模块复选框，重启 QQ 即可。如果 QQ 加了白名单，在最近任务清理可能没用，重启 QQ 步骤-强制停止 QQ 运行-打开 QQ。

  1. 模块需要适配才可以使用，升级应用 QQ 前请查看模块的升级日志是否适配了 QQ 新版本。

  2. 太极各版本图示如下：

太极·阴升级应用小教程

因为太极·阴更改了应用签名，通过应用商店更新将会安装失败，可通过以下教程更新。

【以 QQ 举例】

  1. QQ 官网获取最新版下载
  2. 打开太极
  3. 点击右下角，弹出的菜单中选择创建应用
  4. 创建应用界面，继续点击位于右下角的+号，选择刚才下载的最新版 QQ.apk，文件路径取决于你用什么浏览器下载的 QQ，例如系统自带的浏览器文件保存路径为/sdcard/Download/，其他浏览器查看设置可获取详细路径
  5. 等待太极签名，签名成功后安装即可…

模块为适配新版本？最新版不好用？
因为太极·阴无法使用核心破解模块，所以无法直接降级安装，只能备份数据卸载重装旧版了

无 root 其他框架

应用转生或其他框架使用方法与太极类似，具体软件通过百度获取。

无 root 使用教程到此结束，以下通通可放弃。

确认安卓版本

首先进入手机设置-关于手机/设备，即可看到手机运行的安卓版本，各国产系统查看步骤大同小异。

安卓 8.1 及以下

如果是 8.1 及以下系统，可以使用原版 xposed，但如果系统为 MIUI 就不行。8.1 + miui 系统 会卡开机动画，请勿作死尝试…

提供 官网链接 和 蓝奏云链接 ，下载 xposed 安装并打开，点击界面中的安装选项，弹出 root 权限框，选择确认即可。安装前建议先下载 卸载包，防止出错情况救砖备用。

重启手机后将是全局 xposed，下载模块后需重启手机以开启模块使用。

原版 xposed 使用教程到此结束，以下通通可放弃。

安卓 9.0 及以上

如果是安卓 9.0 及以上，或者是 8.1 + miui 系统，可以看接下来的教程，使用太极·阳或者 EDXposed。

下载 Magsk 模块并添加

添加应用和模块

一切完成，打开太极·阳或者 EDXposed，创建/添加应用程序，比如 QQ，然后下载模块，比如 QX 模块，进入模块管理，勾选 QX 模块复选框，重启 QQ 即可。如果 QQ 加了白名单，在最近任务清理可能没用，重启 QQ 步骤-强制停止 QQ 运行-打开 QQ。【EDXposed 需要在设置中开启即时模块列表才可免重启手机，只需重启应用】
