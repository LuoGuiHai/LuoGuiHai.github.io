安卓手机获取 ROOT 权限

发表于2020-08-02|更新于2021-03-10|ANDROID 教程
|字数总计:1.8k|阅读时长:5分钟|阅读量:

蓝奏云链接可能跟不上官网最新版

前言

root 权限，类似于 Windows 系统中的 Administrator【管理员】，root 是 Linux 和 unix 系统中的超级管理员用户帐户，该帐户拥有整个系统至高无上的权力，所有对象他都可以操作。获得 root 权限之后就意味着已经获得了系统的最高权限，这时候你可以对系统中的任何文件（包括系统文件）执行所有增、删、改、查的操作。

安卓【Android】是一种基于 Linux 内核（不包含 GNU 组件）的自由及开放源代码的操作系统。安卓手机 root 权限是 andriod 的使用最高权限，获取了 root 权限，可以轻松的对手机刷机备份还原卸载系统文件等重要操作。

iPhone 的操作系统【ios】获取 root 权限的操作一般叫越狱。

一般手机获取 root

像小米（Xiaomi）、魅族（Meizu）、一加（OnePlus）等品牌手机，厂商提供了官方通道，获取 root 权限就很简单了。

二. 安装手机驱动

让电脑读取到手机，一般需要为电脑安装手机驱动程序。

  1. 以小米手机为例，下载官方 MiFlash 工具 并解压；

三. 解 BL 锁（Unlock）

为了避免在应用 Android 平台时的一些潜在风险，厂商的通用做法就是设置 BootLoader 锁，防止用户对手机系统进行更改，这种做法在保证安全的同时，一定程度上也限制了 Android 系统潜能的发挥。BootLoader 锁，简称 BL 锁，就是在操作系统内核运行之前运行的一段小程序。

如果你的手机没有 BL 锁，可直接跳过本步骤。

  1. 在需要解锁的设备中登录已经具备解锁权限的小米账号，并进入“设置 -> 开发者选项 -> 设备解锁状态”中绑定账号和设备；

  2. 绑定成功后，手动进入 Bootloader/Fastboot 模式（关机后，同时按住开机键和音量下键）；

  3. 小米解 bl 锁官方网站：http://www.miui.com/unlock/index.html

  1. 解压下载的文件后，双击运行 miflash_unlock.exe；

  1. 点击同意，登录与绑定手机相同的账号登录。按提示操作即可解除 bl 锁…

【小米解锁会清除全部数据，注意数据备份。】

【遇到问题？ 点此查看 FAQ >> 】

四. 下载刷入 recovery【恢复模式】

recovery，简称 rec。Recovery 是 Android 手机备份功能，指的是一种可以对安卓机内部的数据或系统进行修改的模式（类似于 windows PE 或 DOS）。在这个模式下可以，对已有的系统进行备份或升级，也可以在此恢复出厂设置。一般出厂的手机都自带了官方 rec。

刷入第三方的 Recovery，将获得更多的功能，并且可以刷入第三方 rom，官方自带则不行。

五. ROOT 安装

广为人知的 root 获取及管理程序有 SuperSU 和 Magisk。前者不支持高版本安卓，后者功能更为强大，且支持到了 Android 10.0，作者也正在测试 Android11.0。

Magisk 官方介绍：

Magisk is a suite of open source tools for customizing Android, supporting devices higher than Android 4.2. It covers fundamental parts of Android customization: root, boot scripts, SELinux patches, AVB2.0 / dm-verity / forceencrypt removals etc.

Github
可为 Android4.2 以上的系统支持 SU【root】、开机启动脚本、SELinux 补丁、AVB2.0 等等等功能。
Magisk 也称面具、脸谱。

其他手机获取 root 权限

官方暂未提供通道解 BL 锁？如华为，OPPO，VIVO 等新出的手机，这个无解。华为很久之前出的老手机似乎可以某宝买解锁码，请自行搜索。【似乎很贵】

像一些非旗舰，不是很知名的手机如何获取 root 权限呢？如有 BL 锁，上 百度 问一问，或许能有方法。

无 BL 锁或已解 BL 锁，第三方 rec 也可以上 百度 问一问，实在无法可以找一键刷机类的 pc 端工具，比如线刷宝，奇兔刷机。
